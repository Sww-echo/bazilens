## 0. 一句话简介

**BaziLens** 是面向**海外华人**的传统中国命理研究 SaaS 工具。核心功能：八字 / 紫微 / 占卜排盘 + AI 流式解读 + PDF 详批报告。需要为 Sprint 1 设计 **9 个页面 + 1 套组件库**。

技术栈是 **React 19 + Vite 7 + TypeScript + Tailwind CSS（建议 v4）+ react-router-dom 7**。状态管理用 Zustand（已存在，不要替换）。i18n 用 react-i18next，所有中文文案用 `t('namespace.key')` 包装。

## 2. 视觉系统

### 配色 — 请输出 3 个方向供选择

#### 方向 A：墨白（最克制，最学术）
- 主：`#1A1A1A` 墨黑 / `#FFFFFF` 宣纸白
- 强调：`#C4A661` 古铜金（仅用于按钮 / 徽章 / 关键命理元素）
- 比例：黑白 90% + 金 10%

#### 方向 B：青墨（东方意境，**首选**）
- 主：`#1A2332` 深青墨 / `#F5F1E8` 米色宣纸
- 强调：`#C8553D` 朱砂红（CTA / 警示） / `#7A9E9F` 淡青（次要）
- 比例：青墨米 85% + 朱砂 15%

#### 方向 C：现代简（最易商业化）
- 主：`#0F0F0F` 近黑 / `#FAFAF7` 暖白
- 强调：`#D4A574` 金米 / `#2C2C2C` 次黑
- 比例：黑白 80% + 金米 20%

### 字体

| 用途 | 中文 | 英文 / 数字 |
|------|------|------------|
| 标题 | 思源宋体 (Source Han Serif) | Inter / Söhne |
| 正文 | 苹方 / 思源黑体 | Inter |
| 命盘干支字符 | 思源宋体 Bold（避免乱码） | — |
| 表格数字 | — | Inter Tabular Numbers |
| 商标 | 思源宋体或自定义字标 | 配套英文版 |

### 间距与圆角

- 间距尺度：`4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 / 96`（Tailwind 标准）
- 卡片圆角：`12px`（中等克制）
- 按钮圆角：`8px`
- 命盘方格：`0` 圆角（保持古籍方正感）
- 头像 / 状态点：圆形

### 图标

- 风格：**线条（lucide-react / phosphor）**，不要面性，不要彩色
- 命理符号：天干 / 地支 / 五行 / 十神 — **直接用思源宋体字符**当图标，不用 emoji
- 加载动画：用毛笔书写卷轴 / 排盘动画，不要旋转 spinner

### 必须避开（Apple 审核红线）

- ❌ 水晶球 / 塔罗牌堆 / 神秘符号 / 占星轮
- ❌ "幸运数字 / 幸运颜色"博彩感视觉
- ❌ 红包 / 抽奖 / 转盘动画
- ❌ 健康 / 疾病预测插画
- ❌ 闪烁星空 / 漂浮粒子（玄学感过重）

## 10. 给 AI 设计工具的额外指引（v0.dev / Lovable / bolt.new）

如果你用 AI 工具直接生成可运行代码，**请遵循**：

```
- 框架：React 19 + Vite 7 + TypeScript 5.9 + Tailwind v4
- 路由：react-router-dom v7（已存在）
- 状态：Zustand（已存在 4 个 store，import from '@/stores/*'）
- API：所有数据走 src/api/* 已封装的函数（不要自己 fetch）
- Hooks：useReading / useReportProgress / useSubscription / useCharts 已存在
- i18n：所有可见字符串用 useTranslation 的 t('key')
- 表单：原生 React + 受控组件（不要引 react-hook-form）
- 校验：内联手写（不要引 Zod）
- 不要安装新依赖（特别是不要 antd / mui / chakra / shadcn 全套）
- 文件路径请按 src/pages/{auth,chart,reading,report,account,upgrade,public,admin}/
- 组件路径请按 src/components/<PascalCase>.tsx
```

可以使用：
- `lucide-react`（图标，已计划引入）
- `clsx` 或 `cn` 工具函数（条件 className）
- `react-markdown`（流式解读渲染，已计划引入）

不要使用：
- shadcn/ui（项目克制，不需要这么多组件）
- Material UI / Ant Design（设计语言不符）
- Framer Motion（除非真的需要复杂动画，CSS 足够）
